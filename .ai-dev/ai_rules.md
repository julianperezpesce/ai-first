## Architecture Rules

### Adapter Pattern
When adding support for a new framework:
1. Create adapter in `src/core/adapters/`
2. Implement `BaseAdapter` interface
3. Register in `adapterRegistry.ts`
4. Add tests in `tests/adapters.test.ts`

### Analyzer Pattern
When adding new analysis:
1. Create analyzer in `src/analyzers/`
2. Export a function that takes file array
3. Return structured results
4. Add tests

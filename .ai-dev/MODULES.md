## src/core/adapters/

Framework-specific adapters that extract context from different tech stacks.

### Pattern
Each adapter implements `BaseAdapter` interface:
```typescript
interface BaseAdapter {
  detect(files: File[]): boolean;
  analyze(files: File[]): AnalysisResult;
}
```

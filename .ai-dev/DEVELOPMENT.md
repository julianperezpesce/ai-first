# Development Guide

> How to set up, develop, and contribute to ai-first.

---

## Prerequisites

- Node.js 18+
- npm

## Setup

```bash
# Clone
git clone https://github.com/julianperezpesce/ai-first.git
cd ai-first

# Install
npm install

# Build
npm run build

# Test
npm run test
```

## Development Workflow

### 1. Create Feature Branch
```bash

```

### 2. Make Changes

### 3. Run Tests

```bash

```

### 5. Commit


## Adding New Framework Support

### 1. Create Adapter
```typescript
// src/core/adapters/community/myFrameworkAdapter.ts
import { BaseAdapter } from '../baseAdapter.js';

export class MyFrameworkAdapter extends BaseAdapter {
  detect(files: File[]): boolean {
    // Return true if framework detected
  }
  
  analyze(files: File[]): AnalysisResult {
    // Extract framework-specific context
  }
}
```

### 2. Register Adapter
```typescript
// src/core/adapters/adapterRegistry.ts
import { MyFrameworkAdapter } from './community/myFrameworkAdapter.js';

adapterRegistry.register('my-framework', new MyFrameworkAdapter());
```

### 3. Add Tests
```typescript
// tests/adapters.test.ts
describe('MyFrameworkAdapter', () => {
  it('should detect framework', () => {});
  it('should analyze code', () => {});
});
```

### 4. Test Against Sample Project
```bash
af init --root test-projects/my-framework-app
```

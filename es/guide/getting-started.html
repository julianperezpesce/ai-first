Primeros Pasos ​

AI-First es una CLI que prepara cualquier repositorio para agentes de codificación de IA.

¿Por qué AI-First? ​

Los asistentes de IA suelen tener dificultades con repositorios grandes:

  • Ventanas de contexto limitadas
  • Falta de conocimiento arquitectónico
  • Dificultad para navegar

AI-First crea una capa de inteligencia del repositorio.

Inicio Rápido ​

bash
# Instalar globalmente
npm install -g ai-first

# Inicializar
ai-first init
